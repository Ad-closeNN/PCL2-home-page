# 由 Ad_closeNN 制作的 PCL 2 主页
### 如何使用
你可以在 PCL 2 设置中选择联网更新，网址是`https://ad-closenn.github.io/aft/Custom.xaml`\
更多内容可访问`https://ad-closenn.github.io/aft/`
